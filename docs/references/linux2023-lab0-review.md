## 為什麼強調「改善中文表達」？

引用李宏毅教授對 ChatGPT 的觀點：有效使用 AI 工具意味著人類的寫作應達到語言模型的品質。工程師必須以精確和清晰作為基本專業能力進行溝通。

**核心原則：** 注意細節是關鍵 — 掌握小的方面才能應對 Linux kernel 三千萬行的程式碼。

## Lab0 預期學習成果

學生應掌握：
- 進階 C 程式設計概念（variable arguments、signal handling、記憶體管理）
- GNU/Linux 開發工具（Cppcheck、Valgrind）
- Git 和 GitHub 工作流程
- 協作用的程式碼標準
- 自動化測試機制
- Linux kernel linked-list 實作與高效排序
